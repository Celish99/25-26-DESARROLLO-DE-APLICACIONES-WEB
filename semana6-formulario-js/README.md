# Formulario con Validaciones Dinámicas en JavaScript

Formulario interactivo con validaciones en tiempo real usando HTML, CSS y JavaScript.

## Validaciones
- Nombre mínimo 3 caracteres
- Correo con formato válido
- Contraseña segura
- Confirmación de contraseña
- Edad mayor o igual a 18 años
